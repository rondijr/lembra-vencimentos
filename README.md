Lembra Vencimentos
Propósito: lembrar vencimentos (RG, CNH, carteirinhas).
Persona: aluno com prazos dispersos.
Foco 1ª execução: avisos locais; sem dados sensíveis.
Paleta: Blue #2563EB , Slate #0F172A , Amber #F59E0B .
Ícone: calendário com sino.

Primeiros passos:
1. Instale dependências: `flutter pub get`
2. Rode o app: `flutter run`
3. Na primeira execução, clique em "+" para cadastrar o primeiro prazo.
